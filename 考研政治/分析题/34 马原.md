# 一、马原辩证关系
**辩证法**就是要用全面的、发展的和联系的观点把握事物的本质和规律，发现矛盾、解决矛盾。矛盾是指事物或事物之间对立统一的关系。矛盾的两种基本属性是同一性和斗争性，是矛盾双方相互联系的两个方面。要求在对立中把握统一，在统一中把握对立，坚持全面地看问题。分析和解决矛盾时，还要坚持两点论与重点论的统一，抓主要矛盾，做到具体问题具体分析。
  1. **第一，相互区别：** A是……(定义)，B是……(定义)。所以他们是相互区别的：写明不同点。
  2. **第二，相互联系：** A中有B，B中有A，AB有共同点(结合材料写)。A和B相互作用，两者相互联系和影响。
  3. **第三，转化与促进：** 在一定条件下相互转化或促进。没有A就没有B。
  4. **总结：** A与B构成矛盾运动，推动事物的发展。
# 二、决定关系相关原理
#A决定B #如生产力和生产关系 #实践和认识 #社会存在和社会意识
- (1) A对B起着决定作用，A是不依赖于B的客观实在。
    
- (2) B是A的反映，并反作用于A。B应当与A相协调发展，当B与A相协调发展时，就能促进人类社会的发展，如果不协调则有阻碍作用。
    
- (3) AB相作用和相互协调要在尊重客观规律的基础上，发挥人的主观能动性，在实践中动态发展，实现A与B的良性互动。
# 三、其他原理
- **主要矛盾：** 矛盾体系中处于支配地位、对事物发展起决定性作用的矛盾。事物的性质是由主要矛盾的主要方面所决定的。
    
- **实践与认识：** 实践是认识的基础，实践在认识活动中起着决定性的作用。坚持实践第一的观点，一切从实际出发，同时也要注重理论对实践的指导作用。实践与认识的辩证运动，是一个循环往复以至无穷的辩证发展过程。
    
- **真理与价值：** 任何成功的实践都是真理尺度和价值尺度的统一，是合规律性和合目的性的统一。
    
- **真理检验标准：** 实践是检验真理的唯一标准，但实践对真理的检验不可能一次完成，实践检验真理是一个永无止境的发展过程。
    
- **人民群众：** 人民群众是历史的主体和创造者。历史是人民群众创造的，但人民群众创造历史的活动及作用又受到社会历史条件的制约。
    
- **科学技术：** 科技是推动社会发展的重要力量，也是一把双刃剑。科学技术能够通过促进经济发展造福于人类，但在运用于社会时所遇到的问题也越来越突出。
